# A Twitter Clone made with React

A twitter clone built to demonstrate the full stack (client, server, database).

* Anyone can send a tweet
* Anyone can see the list of all of the tweets sent

## Front-end

* React 
* Bootstrap CSS

## Back-end

* NodeJS
* Express
* Morgan
* Cors
* Monk (MongoDB)
* Bad-Words
* Express-rate-limit
